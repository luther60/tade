Exemples concrets

Etude de cas

Tableau électrique à remplacer

  • Problème : tableau ancien, protections insuffisantes, repérage absent.
  • Risque : sécurité + pannes + conformité floue.
  • Solution : remplacement + repérage circuits + protections adaptées.
  • Résultat : tableau lisible, circuits identifiés, installation sécurisée.

Borne de recharge (IRVE)

  • Problème : besoin quotidien, puissance à définir.
  • Risque : sous-dimensionnement, protections mal choisies.
  • Solution : ligne dédiée + protections + installation conforme.
  • Résultat : charge stable, sûre, adaptée à l’usage.

Collectivités : relamping / maintenance

  • Problème : éclairage vieillissant, pannes, consommation.
  • Risque : sous-dimensionnement, protections mal choisies.
  • Solution : remplacement luminaires, optimisation, planification.
  • Résultat : continuité de service + baisse maintenance + plan clair.

Questions fréquentes

Intervenez-vous le week-end ? Intervention
Selon disponibilité et criticité (sécurité / panne). Le plus simple : appelez, on vous dit immédiatement.
Pourquoi le déplacement est facturé ? Déplacement
Parce qu’on vient préparé, pas “pour regarder”. Moins d’allers-retours, plus d’efficacité, meilleur résultat.
Donnez-vous un devis avant travaux ?Devis
Oui. Pour les travaux planifiés : devis structuré (matériel, main d’œuvre, options, délais).
Travaillez-vous pour syndics / commerces / collectivités ?Clients
Oui. Avec devis clair, planification et interventions propres (priorité continuité de service).
“Mise aux normes”, ça veut dire quoi ?Normes
Vérification des protections, conformité du tableau, sécurité des circuits, correction des points à risque. On explique ce qui est nécessaire vs optionnel.
Proposez-vous un SAV ?SAV
Oui. Si un réglage ou contrôle est nécessaire après intervention, on suit le dossier.
Comment préparer l’intervention ?Post-intervention
Si possible : photo du tableau, description du problème, adresse + code postal. Ça fait gagner du temps.
